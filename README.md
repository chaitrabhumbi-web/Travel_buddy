# Travel Buddy Website

## Description
Travel Buddy is a responsive web application designed to help users explore travel destinations easily. The application provides a clean and interactive interface for discovering places and planning trips.

This project was developed using HTML, CSS, and JavaScript with AI-assisted development (Claude), and further customized to improve functionality and user experience.

## Features
- Responsive design for mobile and desktop
- Interactive and user-friendly interface
- Organized and modular code structure
- Easy navigation for travel exploration

## Tech Stack
- HTML
- CSS
- JavaScript
